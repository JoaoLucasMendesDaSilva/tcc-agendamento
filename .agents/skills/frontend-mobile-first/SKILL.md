# Skill: Frontend Mobile-First

## Regras

- Mobile-first.
- Textos em português.
- Poucos campos por tela.
- Botões grandes.
- Fluxo simples.
- Feedback de loading, erro e sucesso.
- Acessibilidade básica.
- Evitar dependências visuais pesadas.

## Telas principais

- Login
- Cadastro
- Dashboard
- Serviços
- Profissionais
- Agenda
- Agendamento público

## Fluxo público

1. Ver negócio.
2. Escolher serviço.
3. Escolher profissional.
4. Escolher data/horário.
5. Informar nome e telefone.
6. Confirmar.

## Resposta esperada

1. Arquivos criados.
2. Como navegar.
3. APIs usadas.
4. Como testar no mobile.
5. Pendências.
